# Session Briefing - INFOTIK 26 (Pusat Informasi Mahasiswa TI 2026 UMKT)

## Project Overview
Website Sistem Informasi Pusat Mahasiswa Teknik Informatika Angkatan 2026 (INFOTIK 26) sebagai "one-stop hub" angkatan untuk meminimalisir pertanyaan berulang di grup WhatsApp, memberikan informasi masta, jadwal kuliah, ujian BTQ, tips akademik/sosial, daftar tugas, forum diskusi, buletin berita, verifikasi grup WA, dan panel admin.

## Current Status
- Initialized Git repository.
- Created `.gitignore` and `.graphifyignore`.
- Creating implementation plan for project structure and features.

## Critical Goals
- Build clean, modern, responsive full-stack/SPA web application using Tailwind CSS, Alpine.js, Lucide Icons, and localStorage persistence.
- Implement all 9 main features + Admin panel with high visual aesthetic standards (indigo/purple theme + gold/emerald accents, dark/light mode toggle, mobile bottom nav + desktop sidebar).
